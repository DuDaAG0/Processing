3.9 Verschiedene Formen Level 1

Dein Programm soll bestimmte Formen zeichnen, wenn bestimmte Tasten gedrückt werden.

Starte mit folgender Grundstruktur:


void setup() {
  size(400, 400);
  background(255);
}
void draw() {
  // Dein Code
}

Dein Programm soll Folgendes ausführen:

1. Wenn die Taste r gedrückt ist, dann zeichne einen roten Kreis

2. Wenn die Taste b gedrückt ist, dann zeichne ein blaues Quadrat.

4. Wenn die Taste x gedrückt ist, dann lösche alle bisherigen Formen (Befehl background).

3.10 Zeichenprogramm Level 1

In dieser Aufgabe soll ein Zeichenprogramm mit verschiedenen Farben erstellt werden. Starte wieder mit folgender Grundstruktur: 


void setup() {
  size(400, 400);
  background(255);
}
void draw() {
  // Dein Code
}

Schritte:

1. Zeichne Kreise mit Durchmesser 15, die dem Mauszeiger folgen.
2. Lass die Kreise nur noch dann zeichnen, wenn die Maustaste gedrückt ist.
3. Entferne mit noStroke(); die schwarze Umrandung der Kreise.
4. Ändere die Farbe der Kreise, wenn die entsprechende Taste gedrückt wird:

    Taste r: rot
    Taste b: blau
    Taste g: grün

5. Wenn die Taste x gedrückt ist, dann lösche alle bisherigen Formen.
6. Ergänze eine zusätzliche Farbe.

3.11 Zeichenprogramm advanced Level 2

Erweitere dein Zeichenprogramm.

  • Das Fenster soll doppelt so breit wie hoch sein (800 × 400).
  • In der linken Hälfte wird mit einem Kreis gezeichnet, wenn die Maus gedrückt ist.
  • In der rechten Hälfte wird mit einem Rechteck gezeichnet, wenn die Maus gedrückt ist.

Hinweis: Du kannst innerhalb einer if-Anweisung weitere if-Anweisungen verwenden. Das nennt man verschachtelte if-Anweisungen.

3.12 Zeichenprogramm synchron Level 3

Ändere dein Programm aus Aufgabe 3.11:

  • Wenn du in der linken Hälfte Kreise zeichnest, sollen gleichzeitig auch in der rechten Hälfte Quadrate gezeichnet werden.
  • Wenn du in der rechten Hälfte Quadrate zeichnest, sollen ebenfalls links Kreise mitgezeichnet werden.

3.13 Zeichenprogramm achsengespiegelt Level 4

Ändere dein Programm aus Aufgabe 3.12 weiter:

Alles, was links mit Kreisen gezeichnet wird, soll auch rechts gespiegelt mit Quadraten erscheinen und umgekehrt.

Interaktive Fläche:

Tipp: Arbeite mit mouseX und überlege, wie du die x-Position spiegeln kannst. Die Größe des Fensters ist dir bekannt.

3.14 Zeichenprogramm punktgespiegelt Level 4

Ändere dein Programm aus Aufgabe 3.13:

  • Das Fenster soll quadratisch sein (800 × 800).
  • Auf der linken Hälfte werden Kreise und auf der rechten Hälfte Rechtecke gezeichnet.
  • Die Zeichnung soll an der mittigen horizontalen Linie gespiegelt werden.
  • Die Zeichnung soll zusätzlich an der mittigen vertikalen Linie gespiegelt werden.
  • Die Zeichnung soll außerdem am Mittelpunkt gespiegelt werden.
