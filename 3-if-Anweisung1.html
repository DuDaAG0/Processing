3.1 Hintergrund wechsel dich Level 1

Interaktive Fläche:

Tipp: Mit background(0); wird der Hintergrund schwarz und mit background(255); weiß.

3.2 Leuchtender Kreis Level 1

Folgendes Programm erzeugt einen Mauszeiger in Form eines roten Kreises. 


void setup(){
  size(300, 300);
}

void draw() {
  background(255); 
  fill(255, 0, 0); 
  circle(mouseX, mouseY, 20, 20);
}

Verändere das Programm so, dass in der rechten Bildschirmhälfte der Mauszeiger ein grüner Kreis wird und in der linken wieder rot.

Interaktive Fläche:

3.3 Leuchtender Kreis mit Spur Level 1

Passe dein Programm aus Aufgabe 3.2 so an, dass der Kreis eine Spur hinterlässt.

Interaktive Fläche:

Tipp: Überlege, welche Anweisung dafür verantwortlich ist, dass der Bildschirm jedes Mal neu gezeichnet wird.

3.4 Mauszeiger mit Formen Level 1

Verändere das Programm aus Aufgabe 3.2 so, dass in der oberen Bildschirmhälfte ein roter Kreis erscheint und in der unteren Bildschirmhälfte eine grüne Box.

3.5 Mauszeiger mit Form und Farbe 1.0 Level 2

Finde heraus, wie sich die Form und Farbe des Mauszeigers in der interaktiven Fläche verändert.

Interaktive Fläche:

Schreibe anschließend ein Programm mit der gleichen Ausgabe.

Tipp: Man kann innerhalb einer if-else-Anweisung eine weitere if-else-Anweisung schreiben.

3.6 Mauszeiger mit Form und Farbe 2.0 Level 2

Passe deinen Code aus Aufgabe 3.5 so an, dass du keine verschachtelten if-Anweisungen mehr verwendest.

Tipp: Wenn zwei Bedingungen gleichzeitig gelten sollen, kannst du sie mit && (UND) verbinden. 


if(xMouse < 100 && yMouse < 100){
  ...
}

3.8 Viertel 2.0 Level 3

Löse die Aufgabe 3.7 erneut, aber verwende nur eine einzige if-else-Anweisung.

Tipp 1: Der ODER-Operator ist ||. Es gilt: der Ausdruck ist wahr, wenn mindestens eine Bedingung erfüllt ist.

Tipp 2: Man kann in logischen Ausdrücken Klammern nutzen wie in der Mathematik.
